# 追番计划小程序

<div align="center">
  <img src="https://gcore.jsdelivr.net/gh/qiyuor2/blog-image/img/2022-7-30zhuifanqrcode.jpg" style="width: 200px" />
</div>

追番列表是一个用来记录个人动画观看情况的小程序。

本项目的所有用户数据均保存在本地，因此当进行更换设备、删除小程序等操作时需要在设置界面导出数据进行备份。
